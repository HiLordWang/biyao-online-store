this is my first e-commerce project
这是一个电商项目，仿造的是必要商城。
使用时需要将project文件夹放在本地服务器中，并在本地服务器中打开。
JS部分主要使用jQuery部分采用原生JS，使用了轮播图以及cookie插件。
商品列表(list.html)页面CSS采用SASS编写。
主页及所以子页面都提供了各页面的链接，点击可跳转到其他页面。


主页(index.html)方面。
rightBar 用position: fixed;固定在右边栏，配合滚动事件，显示消失。点击实现回到顶部及弹出框功能。
nav-clearfix 开始时隐藏，滚动到一定距离后显现，并一直固定在顶部。点击“全部分类”会有二级菜单，二级菜单中有三级菜单。
banner是一个轮播图，采用自己封装的轮播图插件实现。
main部分采用AJAX请求JSON数据，实现内容加载，并还原原网站下拉加载更多功能，用滚动事件配合AJAX实现，加载两次后停止。

商品列表页面(list.html)整体布局与主页相仿，具备主页的公用功能。
main部分采用AJAX请求JSON数据自动生成。
在登录的情况下点击“加入购物车”按钮可实现将当前商品添加至购物车功能，使用cookie实现。
未登录的情况下，点击无效果。

商品详情页面(item.html)整体布局与主页相仿，具备主页的公用功能。
section部分点击右侧小图可改变商品部分大图，点击“体验3D模型”按钮会出现3D展示界面，通过点击360度查看商品。
点击“体验眼镜验光”验光则会弹出模态框，出现二维码。

注册页面(accLog.html)

登录页面(login.html)后台链接了PHP后台，模拟了一个账号(账号：13852171360；密码：123321a;)，验证成功后设置会话cookie，实现登录功能。
勾选下次自动登录后，可将cookie有效期设置为7天。

购物车页面(shop.html)有三种页面表现形式，用户未登录，用户已登录未购物，用户已登录已购物三种不同状态会显示三种界面。
在商品列表页面添加的商品会在购物车页面渲染，通过cookie配合AJAX实现。
购物车有计算单样商品总结及全部商品总价的功能。
购物车有全选功能，勾选后选中所有商品，点击删除则会删除所有商品，并删除cookie。
购物车有增加减少商品数量功能，通过点击“+”“-”功能实现，增减功能也是通过改变cookie实现，改变时，商品总价，及全部商品总价均会一起改变。
当减少到最后一件商品时再次点击“-”按钮，会弹出模态框，告知用户商品只剩最后一件，不能再减少。
点击“结算”按钮跳转到设定的页面。
